HTML

HTML(하이퍼텍스트 마크업 언어, HyperText Markup Language)은 웹 페이지를 만드는 데 사용되는 언어야.
웹 브라우저가 HTML을 해석해서 우리가 보는 웹사이트를 화면에 표시해 줘.

CSS

CSS는 웹 페이지의 디자인과 스타일을 지정하는 언어야. HTML이 구조(내용)를 담당한다면,
CSS는 그 내용을 예쁘게 꾸며주는 역할을 해!

JAVASCRIPT

JavaScript(JS)는 웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어야!
HTML이 구조, CSS가 디자인을 담당한다면, JavaScript는 기능을 추가하는 역할을 해.
